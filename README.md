## 💻 Sobre o projeto

<!-- EXPLICA O MOTIVO DO PROJETO -->
Este é um projeto que estou desenvolvendo para a criação do meu portfólio pessoal, onde irei hospedar meus projetos, aplicações web e compartilhar minha jornada na programação.

Me inspirei no projeto de portfólio desenvolvido pelo Rodolfo, do canal DevClub no YouTube, que tem sido uma grande referência nesse processo.

Atualmente, o projeto está com aproximadamente 10% concluído. A cada etapa finalizada, irei documentar e compartilhar aqui no repositório a evolução, melhorias e aprendizados adquiridos durante o desenvolvimento.

Este portfólio não é apenas um projeto visual, mas também um espaço onde poderei demonstrar minhas habilidades, práticas de desenvolvimento e crescimento contínuo como desenvolvedor.

## 💪 Como contribuir para o projeto

1. Faça um **fork** do projeto.
2. Crie uma nova branch com as suas alterações: `git checkout -b my-feature`
3. Salve as alterações e crie uma mensagem de commit contando o que você fez: `git commit -m "feature: My new feature"`
4. Envie as suas alterações: `git push origin my-feature`
> Caso tenha alguma dúvida confira este [guia de como contribuir no GitHub](./CONTRIBUTING.md)
